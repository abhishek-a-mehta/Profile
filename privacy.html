Privacy Policy

Do i collect your data ?

No, I don't collect any of the user data. This tools are developed to help fellow Admins to help perform their day to day tasks with ease. This tools are developed with the help of Google AppScript which runs on V8 Engine which is the most secure JavaScript Engine.

Whatever data you download or upload using this tools directly goes to Google Workspace Admin Console using required APIs to perform certain actions (Like add/Delete/Suspend/Update - Users/Groups/Attributes) and upon successfully completing the action it gives confirmation to the user.

All of the above is done using a secure pipeline of OAuth and makes sure that data reaches the Specific Intended user only (Who is using this tool)

Can i see or access your data ?

No, I can not access or see your data as data is stored in the google sheet under your ownership and unless you specifically add any user to that sheet not only me but no one can see your data.
